    • Comunicación No Verbal

    La comunicación no verbal complementa y, a veces, contradice la comunicación verbal. Incluye:

    • Lenguaje Corporal: Postura, gestos, expresiones faciales.
    • Contacto Visual: Mantener un contacto visual adecuado para mostrar interés y confianza.
    • Espacio Personal: Respetar la distancia física entre interlocutores.
    • Apariencia: La forma en que nos vestimos y nos presentamos.

    Ejercicios de Comunicación No Verbal

    Para mejorar la comunicación no verbal, se pueden practicar ejercicios como:

    • Observación de videos sin sonido para interpretar emociones.
    • Práctica frente a un espejo para ajustar posturas y gestos.
    • Juegos de mímica.

    Escucha Activa

    La escucha activa es la capacidad de escuchar y comprender completamente el mensaje de otra persona. Implica:

    • Prestar Atención: Concentrarse en lo que dice el interlocutor.
    • Clarificar: Hacer preguntas para asegurar la comprensión.
    • Parafrasear: Repetir lo que se ha entendido para confirmar.
    • Empatía: Intentar comprender los sentimientos y perspectivas del otro.

    Ejercicios de Escucha Activa

    Para mejorar la escucha activa, se pueden practicar ejercicios como:

    • Resumir conversaciones.
    • Juegos de "teléfono descompuesto" con enfoque en la precisión.
    • Entrevistas simuladas.
